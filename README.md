# ConectaMS 🌱

**Plataforma web gratuita para descobrir, apoiar e divulgar projetos sociais de Campo Grande, MS.**

🔗 **Acesse o site:** [centurykun.github.io/conectams.github.io](https://centurykun.github.io/conectams.github.io)

---

## 📌 Sobre o projeto

O **ConectaMS** é uma plataforma desenvolvida como **Atividade Extensionista II – Tecnologia Aplicada à Inclusão Digital (Projeto)** do curso de **CST em Análise e Desenvolvimento de Sistemas** no **Centro Universitário Internacional UNINTER**.

O objetivo é conectar voluntários, doadores e parceiros às ONGs e associações comunitárias ativas em Campo Grande e região, promovendo visibilidade digital gratuita para o terceiro setor local.

---

## 🎯 Objetivos

- Desenvolver uma aplicação web responsiva para divulgação de projetos sociais locais, utilizando **HTML, CSS e JavaScript**
- Configurar um ambiente de hospedagem seguro via **GitHub Pages**
- Mapear adequações necessárias para conformidade com a **LGPD** e o **Código de Defesa do Consumidor**

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura semântica
- **CSS3** — estilização e responsividade
- **JavaScript (ES6+)** — interatividade e animações
- **Formspree** — envio de formulários sem back-end
- **GitHub Pages** — hospedagem gratuita e contínua
- **Google Fonts** — tipografia (Playfair Display + DM Sans)

## ✨ Funcionalidades

- **Hero responsivo** com chamada para ação
- **Seção "Como funciona"** com os 3 passos da plataforma
- **Cards de projetos** com modais detalhados de cada ONG
- **Formulário de cadastro** de organizações integrado ao Formspree
- **Menu mobile** com hamburguer e animação de entrada ao rolar a página
- **Header com sombra dinâmica** ao fazer scroll
